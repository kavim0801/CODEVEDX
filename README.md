# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using Python.

The analysis focuses on understanding the relationship between study hours, attendance, and academic scores. The project also identifies patterns and trends in student performance using data analysis and visualization techniques.

## Objectives

- Analyze student academic performance.
- Check and clean the dataset.
- Identify missing values and duplicate records.
- Calculate average student scores.
- Analyze the relationship between study hours and performance.
- Analyze the relationship between attendance and performance.
- Compare Mathematics and English scores.
- Identify the top-performing students.
- Present findings using charts and graphs.

## Dataset

The dataset contains information about 20 students.

## Data Cleaning

The following data quality checks were performed:

- Checked the dataset structure.
- Checked column names.
- Checked data types.
- Checked for missing values.
- Checked for duplicate rows.
- Removed duplicate records if present.
- Converted numerical columns to appropriate data types.

## Analysis Performed

The following analysis was performed:

1. Calculated the average Mathematics score.
2. Calculated the average English score.
3. Calculated the overall average score.
4. Calculated average study hours.
5. Calculated average attendance.
6. Identified the top 5 performing students.
7. Analyzed the relationship between study hours and average score.
8. Analyzed the relationship between attendance and average score.
9. Compared Mathematics and English scores.
10. Performed correlation analysis.

## Visualizations

The project includes:

- Study Hours vs Average Score
- Attendance vs Average Score
- Math Score vs English Score
- Correlation Heatmap

## Key Insights

- Students with higher study hours generally tend to have higher average scores.
- Higher attendance is generally associated with better academic performance.
- Mathematics and English scores show a positive relationship.
- The top-performing students can be identified using their average scores.
- Study habits and attendance can be important factors related to academic performance.
